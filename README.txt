Chrome version: Version 92.0.4515.159 (Official Build) (x86_64)
OS: MACOS Big Sur version 14. Note: It also runs on windows
Automation framework used: Cypress
Additional libraries used: cypress-file-upload

Requirements: Have node installed
https://nodejs.org/es/download/

How to run a cypress code:
Inside the root directory inside the file execute the following command:
-cd Automation
-npx cypress open

Troubleshooting:

How to install cypress(First time setup):
-npm init
-npm install cypress --save-dev

How to install the library cypress-file-upload:
-npm install --save-dev cypress-file-upload
